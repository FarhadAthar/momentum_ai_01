# Momentum-AI Premium Mobile App UI Design Prompt

Create a production-level premium mobile app UI design for an AI-powered productivity, goals, habits, and personal growth app named “Momentum-AI”.

Use the provided Momentum-AI logo exactly as the official brand logo. Do not redesign the logo. Place the logo naturally in the splash screen, onboarding screens, login screen, and dashboard header where appropriate.

The app must feel like a polished real startup product, ready for Flutter development and Figma handoff.

## Design Direction

Create a modern light theme mobile app using Material 3 design principles.

The UI should feel:
- Premium
- Clean
- Modern
- Human
- Motivational
- Calm but energetic
- Professional
- App-store ready
- Non-robotic
- Non-generic
- Production-level

Avoid a boring AI chatbot look. This is not just a chatbot app. It is a complete AI-powered personal momentum and productivity platform.

## Visual Style

Use a premium light theme with soft depth.

Style should include:
- Material 3 components
- Soft rounded cards
- Elegant shadows
- Smooth elevation
- Clean white and off-white backgrounds
- Soft blue, indigo, violet, and teal accents
- Subtle gradients
- Beautiful spacing
- High-quality dashboard layout
- Modern mobile UX
- Premium wellness/productivity feel

Use neumorphism only lightly where it improves the design, for example:
- Soft progress cards
- Momentum score card
- Small action buttons
- Habit streak widgets

Use glassy/glassmorphism touch only where appropriate, for example:
- AI insight card
- Floating dashboard cards
- Highlight cards

Do not overuse neumorphism or glassmorphism. Keep the design clean, usable, and professional.

## Color Palette

Use a light premium palette:

Background:
- Warm white
- Soft off-white
- Very light gray

Primary accents:
- Electric blue
- Indigo
- Soft violet
- Teal/cyan

Text:
- Deep navy or charcoal for headings
- Medium gray for body text
- Muted gray for secondary text

Cards:
- White cards
- Soft gradient cards
- Light glass cards
- Subtle border cards

Avoid:
- Heavy black backgrounds
- Cheap neon overload
- Robotic blue-only AI theme
- Too many random colors
- Low contrast text
- Cluttered layouts

## Typography

Use clean modern typography similar to premium apps.

Typography style:
- Strong bold headings
- Clean readable body text
- Large friendly dashboard numbers
- Clear labels
- Good hierarchy
- Mobile-friendly font sizes

The text must look polished, not like a template.

## UX Direction

Design the app like a real product.

Focus on:
- Easy navigation
- Clear user flow
- Strong visual hierarchy
- Comfortable spacing
- Simple actions
- Beautiful cards
- Quick understanding
- Minimal cognitive load
- Friendly AI interaction
- Motivational progress experience

The app should feel helpful and premium, not complex or technical.
