# Momentum-AI - Stitch Premium Mobile App Design Prompt

Create a premium production-level mobile app UI design for an AI-powered productivity and personal growth app named “Momentum-AI”.

The app helps users build daily momentum by managing goals, habits, progress, and receiving AI-powered guidance.

The design should feel premium, modern, energetic, human, and motivational. It should not look robotic, generic, boring, or like a basic chatbot app.

Use inspiration from premium fitness/productivity apps like Muscle Monster, but do not copy any existing app. Create an original design system for Momentum-AI.

## App Style

Design style:
- Premium dark mobile UI
- Modern glassmorphism cards
- Bold typography
- Strong spacing
- Rounded premium cards
- Smooth gradients
- Energetic but clean layout
- Human-focused AI experience
- Motivational dashboard
- Professional production-ready interface

## Color Direction

Use a dark premium theme.

Main colors:
- Deep black / dark navy background
- Electric blue accents
- Neon purple highlights
- Soft cyan glow
- White text
- Muted gray secondary text
- Subtle gradient cards

Avoid:
- Flat white background
- Cheap template colors
- Too many random colors
- Overly robotic AI blue-only design

## Typography

Use bold, modern, premium typography.

Text should feel:
- Clean
- Confident
- Motivational
- Easy to read
- Mobile-friendly

Use strong headings and clean body text.
