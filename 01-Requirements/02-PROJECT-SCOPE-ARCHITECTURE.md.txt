## 1. Project Vision

Momentum-AI is a premium AI-powered mobile application designed to help users build momentum in their daily goals, planning, habits, productivity, and personal progress.

The app should feel modern, premium, clean, energetic, and human. It should not feel robotic or generic.

Design inspiration can be taken from the Muscle Monster app style, but the final UI must be original.

---

## 2. Design Direction

### Overall Feel
- Premium mobile app
- Dark modern UI
- Energetic and bold visual style
- Smooth cards and sections
- Human-focused experience
- Non-robotic AI feel
- Clean layout with strong spacing

### Visual Style
- Dark background
- Premium gradient highlights
- Glassmorphism cards where needed
- Bold typography
- Rounded cards
- Modern dashboard layout
- Minimal but powerful icons
- Smooth onboarding screens

### Avoid
- Boring AI chatbot look
- Too much plain white UI
- Overloaded screens
- Cheap template-style design
- Robotic AI visuals
- Random colors

---

## 3. Technology Stack

### Mobile App
Framework: Flutter  
Language: Dart  
State Management: Riverpod  
Routing: GoRouter  
Local Storage: SharedPreferences / Flutter Secure Storage  
HTTP Client: Dio  
Architecture: Feature-based clean structure  

### Backend
Runtime: Node.js  
Framework: Express.js  
Database: PostgreSQL  
ORM: Prisma  
Authentication: JWT-based auth  
Password Security: bcrypt  
Validation: Zod / Joi  
File Uploads: Cloudinary or AWS S3  
API Style: REST API  

### AI Integration
AI Provider: OpenAI API or selected AI model  
AI Usage:
- Smart recommendations
- Goal guidance
- Personalized suggestions
- AI assistant/chat if required later

### Admin Panel
Optional for later phase.
